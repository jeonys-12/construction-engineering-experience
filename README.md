# 건설기술인 국외경력 신고 안내

해외사업부문 기술인이 한국건설기술인협회에 국외경력을 신고할 때 필요한 절차, 양식 및 증빙서류를 안내하는 정적 웹페이지입니다.

## 주요 기능

- 신고 절차와 필수서류 안내
- ㈜한화 / ㈜한화건설 기간별 HWP 양식 다운로드
- 출입국일 기준 국외경력 인정기간 계산
- 브라우저에만 저장되는 제출 전 체크리스트
- 모바일·키보드 접근성 대응

## 개인정보 안내

이 페이지는 문서 업로드나 개인정보 수집 기능을 제공하지 않습니다. 작성 자료는 회사가 승인한 보안 전달 경로를 이용해야 합니다.
